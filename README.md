# Privacy Policy Analyzer 🔍

A Chrome extension that analyzes website privacy policies and highlights risky clauses, data sharing practices, and readability level.

## Features
- Detects risky privacy clauses
- Calculates readability score (Flesch Reading Ease)
- Shows privacy risk percentage
- Highlights data sharing and sensitive data collection
- Provides a summary of policy risks

## Technologies Used
- JavaScript
- HTML
- CSS
- Chrome Extension Manifest V3

## Installation

1. Download the repository
2. Open Chrome
3. Go to `chrome://extensions`
4. Enable Developer Mode
5. Click **Load Unpacked**
6. Select the project folder

## How it Works

The extension scans privacy policy text on webpages, detects risky keywords and clauses, and displays a summarized risk analysis.
